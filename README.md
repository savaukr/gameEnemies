ІНСТРУКЦІЯ ДЛЯ ЗАПУСКУ:
1. Клонуйте репозиторій 
        https https://github.com/savaukr/gameEnemies.git
        ssh   git@github.com:savaukr/gameEnemies.git  
    команда для клонування git clone [addr]
2. перейдіть в папку з клонованим репозиторієм
     запустіть по черзі команди
     npm i
     npm run dev
   після цього в браузері відкриється автоматично вкалдка http://localhost:8081/ з грою
